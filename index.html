The backend is built using FastAPI and exposed via NGINX reverse proxy configured with SSL/TLS certificates generated using Certbot.

It leverages Docker Compose to orchestrate multi-container environments. All components are containerized and deployed on dynamically created Google Cloud VM instances.

Repositories are validated server-side to ensure compliance with required structure, including `docker-compose.yml`.
